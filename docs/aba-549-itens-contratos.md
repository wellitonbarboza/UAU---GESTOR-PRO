# Referência da Aba 549 - Itens dos Contratos

Este documento resume como interpretar a aba **549 (Itens dos contratos)** das planilhas de referência para cadastro e importação.

## Estrutura das Colunas
- **E**: Código do contrato.
- **F**: Objeto do contrato (descrição do contrato).
- **K**: Código do fornecedor/prestador.
- **L**: Nome/descrição do fornecedor.
- **M**: Numeração e código dos itens dentro do contrato (um contrato pode ter múltiplos itens).
- **O**: Descrição dos itens vinculados ao contrato.
- **P**: Unidade de referência de cada item (ex.: kg, m, un).
- **Q**: Quantidade contratada de cada item.
- **R**: Preço unitário do item.
- **S**: Subtotal do item (`quantidade × preço unitário`).
- **G**: Valor total do item (deve coincidir com o subtotal).
- **X**: Quantidade medida/acompanhada de cada item.
- **Y**: Valor medido/acompanhado do item.
- **Z**: Quantidade ainda a medir/acompanhada (saldo físico).
- **AA**: Valor ainda a medir (saldo financeiro do item).

## Observações
- A coluna **H** não é utilizada.
- Use sempre esta aba como referência para manter o vínculo correto entre contratos, fornecedores e seus respectivos itens, incluindo
  unidades, quantidades, preços e saldos a medir.
